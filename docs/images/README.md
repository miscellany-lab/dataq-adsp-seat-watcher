# README Image Assets

Place the validation screenshots used by the main README in this directory.

Expected files:

- `zero-seat-no-alert.png`: terminal log showing repeated `팝업 OCR 확인: 0건` when all seats are 0.
- `seat-detected-terminal.png`: terminal log showing a detected remaining-seat case.
- `telegram-seat-alert.png`: phone screenshot showing Telegram seat alerts.

These files are intentionally separated from OCR debug text logs. Do not include Telegram bot tokens, chat IDs, or private account details in screenshots.
